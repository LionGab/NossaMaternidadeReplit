# Prompts de Otimização

Coleção de prompts especializados para otimização do código do Nossa Maternidade.

## Índice

- **INDEX_PROMPTS.md**: Índice geral de todos os prompts
- **PROMPT_1_DESIGN_SYSTEM.md**: Otimizações do design system
- **PROMPT_2_LOGGER.md**: Sistema de logging
- **PROMPT_3_FLATLIST.md**: Performance de listas
- **PROMPT_4_TESTING.md**: Estratégias de testes

## Como Usar

Estes prompts servem como templates para solicitar otimizações específicas ao Claude Code.

Exemplo:

```bash
# Copiar conteúdo de um prompt
cat docs/prompts/PROMPT_1_DESIGN_SYSTEM.md

# Usar como referência ao pedir otimizações
```
